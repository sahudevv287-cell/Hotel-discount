# Hotel Discount Calculator

A beginner-friendly Python program that calculates the final price of a hotel 
room booking after applying a discount based on the customer type and room type.

## Discount Rules
- **Regular customers:** 10% off all rooms
- **Members:** 15% off standard rooms, 20% off deluxe rooms
- **VIP customers:** 30% off all rooms

## How it works
The user enters:
1. Customer type (`regular`, `member`, or `vip`)
2. Room type (`standard` or `deluxe`)
3. The base price of the room

The program calculates and prints the final price after the appropriate 
discount is applied.
